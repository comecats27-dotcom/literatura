# LiterAlura 📚

Proyecto desarrollado como parte del challenge de **Spring Boot** del programa ONE de Alura.

## Descripción

Aplicación de consola que permite consultar libros desde la API de Gutendex y almacenarlos en una base de datos PostgreSQL.

## Tecnologías utilizadas

* Java 17
* Spring Boot
* Spring Data JPA
* PostgreSQL
* Maven
* Jackson

## Funcionalidades

* Buscar libro por título
* Listar libros registrados
* Listar autores registrados
* Listar autores vivos en un año determinado
* Listar libros por idioma

## API utilizada

Gutendex API

https://gutendex.com/

## Ejecución del proyecto

1. Clonar repositorio
2. Configurar base de datos PostgreSQL
3. Ejecutar proyecto Spring Boot

## Ejemplo de uso

```
1 - Buscar libro por título
2 - Listar libros registrados
3 - Listar autores registrados
4 - Listar autores vivos en determinado año
5 - Listar libros por idioma
```
